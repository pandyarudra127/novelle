# Feature Prioritization

## Priority Framework

Features are classified into three tiers:

- **Must Have (MVP)** — The product cannot launch without these. Core value proposition.
- **Should Have (v2)** — Important for retention and engagement. Adds depth to the experience.
- **Nice to Have (v3+)** — Delighters and differentiators. Valuable but can wait.

---
